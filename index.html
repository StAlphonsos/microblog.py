microblog.py by seanh

A command-line microblogging client in Python 

~ % mb lscommands
You must supply a command name as argument. The available commands are:
lscommands
lspublic [term]
lspersonal [username]
lsprofile [username]
lsreplies
send [message]
msg [username] [message]
lsmsgs
lsgroup 
lsmembers 
join 
leave 
lsfollowers
lsfollowing [username]
follow 
unfollow 
fav <#id>
unfav <#id>
lsfavs [username]
lsmentions
search 
lsfeatured
lsusers 
See --help for detailed instructions

Dependencies

Python 2.6, python-twitter and simplejson (both ship with microblog.py).

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/seanh/microblog.py

get the source code on GitHub : seanh/microblog.py
